# 🚀 Quick Start - Deploy Your Website in 5 Minutes!

## Easiest Method: Netlify Drag & Drop

### Step 1: Prepare Your Files
All your files are ready in the current folder:
- ✅ index.html
- ✅ styles.css
- ✅ script.js
- ✅ README.md

### Step 2: Deploy to Netlify (No Account Needed!)

1. **Open Netlify Drop**
   - Go to: https://app.netlify.com/drop
   - You'll see a drag & drop zone

2. **Drag Your Folder**
   - Select ALL files in your project folder
   - Drag them to the Netlify drop zone
   - Wait 30 seconds for upload

3. **Done! 🎉**
   - Your website is now LIVE on the internet!
   - Netlify will give you a URL like: `https://amazing-site-123.netlify.app`
   - Share this URL with anyone!

### Step 3: Customize Your URL (Optional)

1. Click "Site settings" on Netlify
2. Click "Change site name"
3. Enter: `amirshahan-portfolio`
4. Your new URL: `https://amirshahan-portfolio.netlify.app`

## Alternative: GitHub Pages (If you know Git)

### Prerequisites
- GitHub account
- Git installed

### Commands to Run:

```bash
# Navigate to your project folder
cd "d:\Kiro Project"

# Initialize Git
git init

# Add all files
git add .

# Commit files
git commit -m "Initial commit - My portfolio website"

# Create repository on GitHub first, then:
git remote add origin https://github.com/AmirShahan/amirshahan-portfolio.git

# Push to GitHub
git branch -M main
git push -u origin main
```

### Enable GitHub Pages:
1. Go to your repository on GitHub
2. Settings → Pages
3. Source: main branch
4. Save
5. Wait 2-3 minutes
6. Your site: `https://amirshahan.github.io/amirshahan-portfolio/`

## 🔄 How to Update Your Website

### If using Netlify Drag & Drop:
1. Make changes to your files locally
2. Go to your Netlify site dashboard
3. Drag the updated files again
4. Done! Changes are live

### If using GitHub Pages:
1. Make changes to your files
2. Run:
   ```bash
   git add .
   git commit -m "Updated portfolio"
   git push
   ```
3. Wait 1-2 minutes
4. Changes are live!
